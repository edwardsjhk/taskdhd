Taskdhd is designed as a list making web app to help people with ADHD by showing the first job to do in large, bold text.

The web app was made partly with help of generative AI.


Issues:
Re-ordering list items is currently only supported in mobile.

Data storage:
All data (your list items) is stored locally in your browser as cookies.
If you clear your cookies it will delete all of your list items on the web app.
